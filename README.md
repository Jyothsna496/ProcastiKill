# 🚀 ProcastiKill

> **Your AI Productivity Companion**

ProcastiKill is an AI-powered productivity companion that helps users overcome procrastination by intelligently planning, prioritizing, and completing tasks before deadlines are missed. Instead of relying on passive reminders, ProcastiKill actively assists users with AI-generated action plans, smart scheduling, productivity coaching, and personalized recommendations.

---

## 🌟 Problem Statement

### The Last-Minute Life Saver

Students, professionals, and entrepreneurs often miss deadlines, assignments, meetings, interviews, bill payments, and important commitments. Existing productivity tools mainly rely on reminders, which are easy to ignore and rarely help users take meaningful action.

**ProcastiKill bridges this gap by transforming reminders into proactive AI-driven task execution.**

## 🧠 How It Works

1. User enters task details.
2. Gemini analyzes:
   - Deadline
   - Priority
   - Remaining time
   - Context
3. AI generates:
   - Action Plan
   - Daily Schedule
   - Productivity Recommendations
4. User tracks progress through Habits, Calendar, and Insights.
5. AI continuously provides guidance and recommendations.
